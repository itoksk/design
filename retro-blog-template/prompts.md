# Retro Blog Design Prompts

## 基本プロンプト

### ブログレイアウト生成
```
Create a Y2K-era blog layout with:
- Table-based layout with 3 columns (sidebar-main-sidebar)
- Fixed width of 1000px centered on page
- Pastel color scheme (pink, blue, lavender)
- Decorative borders using CSS ridge, groove, inset styles
- Header with sparkly background pattern
- Multiple animated GIF decorations
- Scrolling marquee text elements
- Custom colored scrollbars
```

### プロフィールカード生成
```
Design a kawaii profile card featuring:
- Rounded avatar with decorative frame
- Glitter or star effects around the image
- Handwritten-style font for username
- Bio section with emoji and emoticons
- Stats display (posts, followers, friends)
- Decorative border with hearts or flowers
- Pastel gradient background
- "Follow Me" button with hover animation
Format:
╔══♡══════♡══╗
║  [Avatar]   ║
║ Name: ゆき  ║
║ Age: Secret♪║
╚══♡══════♡══╝
```

### ブログ記事スタイル
```
Create blog post styling with:
- Date header with decorative symbols (♪ 2024.03.14 Friday ♪)
- Title with colorful prefix/suffix decorations
- Category tags in brackets 【カテゴリー】
- Content with increased line height for readability
- Inline emoji and emoticons throughout text
- "Read More" link with arrow decoration
- Footer with comment count, likes, and reblog stats
- NEW! animated GIF for recent posts
```

### デコレーション要素
```
Generate Y2K web decorations including:
- Animated sparkle cursors
- Falling sakura petals or snow effects
- Rainbow text with color cycling
- Blinking text for important notices
- Scrolling background patterns (stars, hearts)
- Glitter trail following mouse movement
- Decorative dividers with symbols
Examples:
☆.。.:*・゜☆.。.:*・゜☆.。.:*・゜
♡ ─ ♡ ─ ♡ ─ ♡ ─ ♡
━━━━━━━━━━━━━━━━━━
```

## レトロWeb要素

### フレームセット
```
Create frame-based layout (古い形式):
- Top frame for header
- Left frame for navigation
- Main frame for content
- Optional right frame
- No-frames fallback
Note: 非推奨だが時代感のため
```

### テーブルレイアウト
```
Design with table-based structure:
- Nested tables for complex layouts
- Spacer GIFs for positioning
- bgcolor and background attributes
- cellpadding and cellspacing
- align and valign attributes
```

### Flash要素（代替）
```
Replace Flash with modern equivalents:
- CSS animations for Flash intros
- JavaScript for interactive games
- HTML5 audio for sound effects
- Canvas for drawing apps
- SVG for vector animations
```

## 使用上の注意

これらのプロンプトは2000年代初期のWeb文化を再現するもので、
現代のWebスタンダードとは大きく異なります。
ノスタルジックな演出や特定の雰囲気作りにのみ使用し、
アクセシビリティやパフォーマンスは二の次となることをご了承ください。
